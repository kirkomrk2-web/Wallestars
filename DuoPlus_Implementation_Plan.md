# План за Имплементация на DuoPlus SMS Автоматизация

## 1. Резюме
Вместо VSPhone, който не поддържа SMS верификация, ще използваме **DuoPlus Cloud Numbers**. Тази платформа позволява наемане на реални (Non-VOIP) номера, които са критични за успешното преминаване на проверките на Wallester.

## 2. Финансова Рамка (Приблизителна)
*   **Цена за номер (SMS):** ~$0.40 - $0.80 на ден (за Non-VOIP номер в САЩ/UK).
*   **Цена за среда (Cloud Phone):** ~$0.15 - $0.20 за една сесия (ако използваме техните емулатори за регистрация).
*   **Общо на акаунт:** **~$0.60 - $1.00**
*   **Ефективност:** Инвестицията в "чист" номер и устройство драстично намалява риска от блокиране, което спестява време и ресурси за повторни опити.

## 3. Техническа Имплементация
Предлагам създаването на специализиран n8n workflow `duoplus-sms-worker.json`, който ще работи като "инструмент" за основната автоматизация.

### Стъпки на работа (Workflow Logic):
1.  **Заявка (Request):** Workflow-ът получава сигнал за нужда от номер.
2.  **Наемане (Order):** Изпраща API заявка към DuoPlus за нов номер (САЩ/UK).
3.  **Слушане (Listen):** Влиза в цикъл на изчакване (polling) на всеки 5-10 секунди.
4.  **Получаване (Receive):** Когато пристигне SMS, извлича OTP кода чрез Regex.
5.  **Връщане (Return):** Подава кода обратно към основния скрипт за регистрация.

## 4. Необходими Действия
1.  **API Key:** Трябва да вземете API Key от настройките на DuoPlus профила.
2.  **Зареждане:** Уверете се, че имате баланс в DuoPlus.
3.  **Настройка:** Импортирайте създадения workflow и попълнете API Key-я.

*Този план гарантира мащабируемост – можете да пускате десетки регистрации едновременно, стига да има бюджет в баланса.*
